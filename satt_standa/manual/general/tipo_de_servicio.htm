Tipo de Servicio

 
A través de esta opción se configura el tipo de servicio contratado con el CL-FARO, para esto debe ingresar a la siguiente ruta Transportadoras > Tipo de Servicio.
 
Aparecerá en pantalla la siguiente ventana, donde debe digitar el nombre o NIT de la transportadora que desea consultar, si no lo tiene, haga clic en el botón Buscar (así listará todas las transportadoras).
 
Ilustración 40: Pantalla de busqueda de servicio
 
Para visualizar o configurar el tipo de servicio de la transportadora haga clic sobre el NIT de la misma.
 
Ilustración 41: Listado de transportadoras encontradas
 
Aparecerá el siguiente formulario donde puede realizar la configuración (o actualización) del tipo de servicio de cada transportadora. Los campos a diligenciar son los siguientes:
 
Ilustración 42: Formulario de Tipo de Servicio de Transportadoras
 
CAMPO
DESCRIPCIÓN
Tipo Servicio
Seleccione el tipo de servicio contratado con FARO: MA - Monitoreo Activo, EAL - Esferas de Asistencia Logística, EAL/MA (servicio combinado).
Servidor
Seleccione en la lista desplegable el servidor con el cual está registrada su URL, lo identifica como se muestra en el siguiente ejemplo (https://ap.intrared.net:444/ap/satt_faro/index.php)
Tiempo para Despachos Nacionales (Min)
Digite el tiempo (en minutos) contratado para realizar el seguimiento.
Tiempo para Despachos Urbanos (Min)
Digite el tiempo (en minutos) contratado para realizar el seguimiento.
Llegada Automática
Esta opción se marca para dar llegada cuando está ubicado en el último punto de llegada.
Activa
Esta opción permite activar el tipo de servicio, si desea inactivarla déjela desmarcada.
Notificar a correos por Agencia
Esta opción permite notificar por correo a las agencias.
Tipo de Servicio
Seleccione el tipo de servicio contratado con FARO, si es Por Despacho o Por Servicio.
